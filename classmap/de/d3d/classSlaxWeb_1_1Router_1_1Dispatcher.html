Find matching Route

Find the matching Route based on the Request Method and Request URI. If no matching route is found, action from the 404 route is returned, if found. If also the 404 Route is not found, the 'RouteNotFoundException' is thrown. Otherwise the matching Route objects action Callable is returned.
