SlaxWeb\Database\Interfaces\Library::select ( string  $table,
array  $cols 
)

Select query

Run a select query against the database and return the result set if it was successful. On error it returns an empty result set. The input array defines a list of columns that need to get selected from the database. If the array item is another array, it needs to hold the "func" and "col" keys at least, defining the SQL DML function, as well as the column name. A third item with the key name "as" can be added, and this name will be used in the "AS" statement in the SQL DML for that column.

Parameters
string$tableTable on which the select statement is to be executed
array$colsArray of columns for the SELECT statement
Returns
